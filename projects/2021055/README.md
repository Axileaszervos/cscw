# ΠΑΡΑΔΟΤΕΟ 3 - ΕΝΑΛΛΑΚΤΙΚΟ ΣΥΣΤΗΜΑ
Σε αυτό το παραδοτέο, απαιτείται η ενασχόληση μας με fantasy consoles, με σκοπό την προσθήκη δικών μας κριτηρίων για τα συστήματα αυτά. 

## [BITMELO](https://bitmelo.com/)
Το `Bitmelo`, είναι μία fantasy console, η οποία έχει αναπτυχθεί σε γλώσσα [Javascript](https://www.javascript.com/). Το site της κονσόλας παρέχει στον χρήστη ένα εύρος από εργαλεία για την ανάπτυξη παιχνιδιού πάνω σε αυτήν. Διαθέτει Code Editor, Sound Editor και Map Editor. Είναι ένα εύχρηστο περιβάλλον ανάπτυξης παιχνιδιών και παρέχει τον χρήστη ποικίλα εργαλεία.

## [MakeCode Arcade](https://arcade.makecode.com/#editor)
Το `MakeCode Arcade`, είναι μια fantasy console η οποία απεικονίζει μια φορητή κονσόλα, εφάμιλλη του [GameBoy](https://el.wikipedia.org/wiki/Game_Boy). Η δημιουργία των παιχνιδιών υλοποιείται μέσω block programming. Πέραν αυτού, ο χρήστης έχει την δυνατότητα να σχεδιάσει την εμφάνιση του χαρακτήρα του καθώς και το περιβάλλον του παιχνιδιού του, μέσω από ειδικό editor που παρέχει η κονσόλα.

## [Fancade](https://www.fancade.com/)
Το `Fancade`, είναι μια πιο εκσυγχρονισμένη fantasy console, η δημιουργία των παιχνιδιών πραγματοποιείται μέσα από την ίδια την εφαρμογή, μέσω έτοιμων assets σε σχήμα blocks. Πιο συγκεκριμένα, τα blocks αποτελούνται από κατηγορίες, υπάρχουν blocks για εντολές κίνησης, δημιουργίας περιβάλλοντος κ.α. Απευθύνεται σε μικρότερες ηλικίες και ο χρήστης μπορεί να φτιάξει ότι του επιτρέψει η φαντασία του.


## ΚΡΙΤΗΡΙΑ
#### Τα παρακάτω κριτήρια επιλέχθηκαν με βάση τις δυνατότητες που παρέχει η πλειοψηφία των fantasy consoles που επέλεξα προσωπικά για μελέτη και με το κατά πόσο αυτά τα εργαλεία είναι χρήσιμα στη ανάπτυξη ενός παιχνιδιού.
 ### Sound Editor
 O sound editor δίνει την δυνατότητα στον χρήστη να δημιουργήσει την δική του μουσική. Ανάλογα με το όργανο που παρέχει το κάθε παιχνίδι για την σύνθεση μουσικής μελωδίας, ο χρήστης μπορεί να φτιάξει όσα μουσικά κομμάτια επιθυμεί για κάθε λειτουργία του παιχνιδιού, όπως για την κίνηση του χαρακτήρα και για το background του παιχνιδιού.
 
 ### Map Editor
 Ο map editor επιτρέπει στον χρήστη να σχεδιάσει εξ ολοκλήρου τον χάρτη του παιχνιδιού του.
 ### Text Editor

## Σύντομη παρουσιάση του Cisco Webex
Tο cisco webex είναι μια πλατφόρμα τηλεδιάσκεψης. Προσομοιώνει την φυσική παρουσία στην τάξη, μέσω της ψηφιακής τάξης και των εργαλείων που διαθέτει. Μέσω αυτής της πλατφόρμας, δίνεται η ευκαιρία στον εκπαιδευτικό να αλληλεπιδράσει και συνεργαστεί με τα παιδιά, με έναν νέο διαφορετικό τρόπο διάδρασης.

![Cisco-Webex](https://user-images.githubusercontent.com/115211756/224569829-f57a6dd8-4a14-487b-998a-da0d569216b1.jpg)


## Σύντομη παρουσίαση του Edmodo
Tο edmodo είναι μια web πλατφόρμα, που παρέχει έναν ασφαλή και εύκολο τρόπο σύνδεσης και συνεργασίας μεταξύ μαθητών και εκπαιδευτικών. Η πλατφόρμα διαθέτει ποικίλα εργαλεία και δυνατότητες όπως η δημοσίευση εκπαιδευτικού περιεχομένου και υλικού.Τέλος, η πλατφόρμα έχει ως στόχο να εξοικειώσει τα παιδιά με τα κοινωνικά δίκτυα, καθώς και τους δείξει και ένα νέο "μονοπάτι" διδασκαλίας.
