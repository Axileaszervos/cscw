## [BITMELO](https://bitmelo.com/)
Το Bitmelo είναι μία fantasy console, η οποία έχει αναπτυχθεί σε γλώσσα javascript. Το site της κονσόλας παρέχει στον χρήστη ένα εύρος απο εργαλεία για την ανάπτυξη παιχνιδιού πάνω σε αυτήν. Πιο συγκεκριμένα, ο χρήστης χρησιμοποιώντας τον map editor, έχει την δυνατότητα να σχεδιάσει και να δημιουργήσει τον δικό του χάρτη για το παιχνίδι του, χρωματίζοντας το κάθε pixel με την διαθέσιμη γκάμα χρωμάτων που υπάρχει διαθέσιμη. Ακόμη, υπάρχει η δυνατότητα επέκτασης του χάρτη απο τον χρήστη, δίνοντας τις ανάλογες συντεταγμένες που επιθυμεί ο ίδιος. Με τον sound editor, ο χρήστης μπορεί να δημιουργήσει την δικία του μουσική, χρησιμοποιώντας τα πλήκτρα ενός πιάνου. Τέλος, υπάρχει και ο code editor, με το οποίο ο χρήστής
